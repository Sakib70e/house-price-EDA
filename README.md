# House Prices - Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on the Kaggle House Prices dataset using Python.

## Project Objectives

* Understand the dataset structure
* Handle missing values
* Analyze skewed distributions
* Perform correlation analysis
* Create visualizations
* Engineer useful features

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Insights

* Overall Quality strongly affects house prices
* Larger living area increases SalePrice
* Neighborhood significantly impacts property value
* Log transformation reduced target skewness
